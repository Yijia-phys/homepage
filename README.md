# yijialog.org — Personal Homepage

> A minimal, full-screen personal homepage with a focus timer and interactive world map.  
> Entirely generated by **[Claude](https://claude.ai)** (Anthropic).

---

## Pages

| File | Description |
|---|---|
| `index.html` | Main homepage — identity card, live clock, site links, wallpaper system |
| `map.html` | Interactive world map with city markers and location info panel |
| `clock.html` | Pomodoro focus timer with collapsed/expanded states |

---

## Features

- **Glassmorphism UI** — frosted glass cards with adjustable opacity, blur, and brightness
- **Wallpaper system** — 5 built-in gradients + custom image upload, persisted via `localStorage`
- **Animated waves** — canvas-based sine wave background, shared across all pages
- **Live clock** — HH:MM display with IANA timezone support, persisted via `localStorage`
- **World map** — equirectangular projection rendered via `d3-geo`, antimeridian-safe, with city hover panels and glitch photo effects
- **Pomodoro timer** — customizable focus/break durations and session count, collapsed pill and expanded ring modes
- **Settings panel** — blur, brightness, overlay opacity, UI opacity, timezone — all persisted locally

---

## Stack

No build tools. No frameworks. Plain HTML + CSS + vanilla JS.

| Library | Version | Usage |
|---|---|---|
| [D3.js](https://d3js.org) | v7 | World map projection (`geoEquirectangular`, `geoPath`, `geoGraticule`) |
| [world-atlas](https://github.com/topojson/world-atlas) | v2 | TopoJSON world geometry (`land-110m.json`) |
| [Google Fonts](https://fonts.google.com) | — | Playfair Display, Crimson Pro, JetBrains Mono |

Everything else — TopoJSON decoding, canvas rendering, wallpaper switching, wave animation, pomodoro engine — is written from scratch in vanilla JS.

---

## Local Storage Keys

All preferences are stored in `localStorage` and shared across pages.

| Key | Description |
|---|---|
| `current-wp` | Active wallpaper ID |
| `user-wps` | User-uploaded wallpaper images (base64, max 8) |
| `display` | Blur, brightness, overlay opacity, card opacity, UI opacity |
| `timezone` | IANA timezone string (e.g. `Asia/Shanghai`) |
| `pomo-focus` | Pomodoro focus duration (minutes) |
| `pomo-break` | Pomodoro break duration (minutes) |
| `pomo-count` | Number of pomodoro sessions |
