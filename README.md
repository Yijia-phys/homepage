# yijialog.org — Personal Homepage

> A minimal, full-screen personal homepage with a focus timer, music player, and interactive world map.  
> Entirely generated by **[Claude](https://claude.ai)** (Anthropic).

---

## Pages

| File | Description |
|---|---|
| `index.html` | Main homepage — identity card, live clock, site links, wallpaper system |
| `map.html` | Interactive world map with city markers, location info panel, and double-click lock |
| `clock.html` | Pomodoro focus timer with music player |

---

## Features

### Shared
- **Glassmorphism UI** — frosted glass cards with adjustable blur, brightness, overlay opacity, and UI opacity
- **Wallpaper system** — 5 built-in gradient presets + custom image upload (max 8), persisted via `localStorage`
- **Animated waves** — canvas-based sine wave background, shared across all pages via `shared.js`
- **Live clock** — HH:MM display with IANA timezone support

### map.html
- **World map** — equirectangular projection via `d3-geo`, with graticule grid and scanline/vignette overlays
- **City markers** — animated pulse for current location, glow for visited cities
- **Info panel** — hover to preview city photo (with glitch effect), description, and date range
- **Double-click lock** — double-click a city to pin the panel; double-click again to release

### clock.html — Focus Timer
- **Pomodoro engine** — customizable focus/break durations and session count
- **Two views** — collapsed pill (top-center) and expanded ring with progress arc
- **Sound alerts** — Web Audio API tones on session end, no external files
- **Keyboard shortcuts** — `Space` start/pause, `R` reset

### clock.html — Music Player
Reads audio files directly from a local folder via the [File System Access API](https://developer.mozilla.org/en-US/docs/Web/API/File_System_Access_API) — no server, no uploads, no copyright issues.

**Supported formats:** mp3, flac, ogg, wav, aac, m4a

**Three display modes:**

| Mode | Description |
|---|---|
| Minimal | A single play/pause icon button in the bottom-right corner |
| Player | Compact bar (expanded view) or full card (collapsed view) with cover, progress, and controls |
| Immersive | Full-screen rotating vinyl disc, Spotify-style bottom bar, centered lyrics |

**Features:**
- Cover art and metadata read from embedded ID3/Vorbis tags via [jsmediatags](https://github.com/nicktindall/jsmediatags)
- LRC lyric sync — place a `.lrc` file next to the audio file (same name); gaps > 4 s auto-labeled `· Music ·`
- Three-line lyric display (prev / current / next) with directional slide-in animation
- Loop modes: sequence → single → shuffle
- Volume control with localStorage persistence
- Music volume auto-ducks to 20% on pomodoro alert, then fades back
- Playlist panel with styled scrollbar
- Keyboard shortcuts — `Q` prev, `W` play/pause, `E` next, `A` cycle mode, `S` toggle lyrics, `D` toggle playlist, `F` cycle loop mode

## Stack

No build tools. No frameworks. Plain HTML + CSS + vanilla JS.

| Library | Version | Usage |
|---|---|---|
| [D3.js](https://d3js.org) | v7 | World map projection (`geoEquirectangular`, `geoPath`, `geoGraticule`) |
| [world-atlas](https://github.com/topojson/world-atlas) | v2 | TopoJSON world geometry (`land-110m.json`) |
| [jsmediatags](https://github.com/nicktindall/jsmediatags) | 3.9.5 | Audio tag reading (cover art, title, artist) |
| [Google Fonts](https://fonts.google.com) | — | Playfair Display, Crimson Pro, JetBrains Mono |

Everything else — TopoJSON decoding, canvas rendering, wallpaper switching, wave animation, pomodoro engine, music player, LRC parser — is written from scratch in vanilla JS.

## File Structure

```
/
├── index.html
├── clock.html
├── map.html
├── shared.js          # Shared UI module (wallpaper, waves, display sliders, settings panel)
└── assets/
    └── photos/        # City photos for map.html (e.g. shenyang.jpg)
```

`shared.js` exposes a `SharedUI` object used by all three pages:

```js
SharedUI.injectStyles()                        // inject shared CSS into <head>
SharedUI.initWallpaper()                       // wallpaper grid + upload
SharedUI.initDisplay(['blur','bright', ...])   // display sliders
SharedUI.initWaves()                           // canvas wave animation
SharedUI.initSettings(panelId, gearSelector)  // returns toggle function
SharedUI.save(key, value)
SharedUI.load(key, default)
```

## Local Storage Keys

All preferences are stored in `localStorage` and shared across pages.

| Key | Type | Description |
|---|---|---|
| `current-wp` | string | Active wallpaper ID |
| `user-wps` | JSON | User-uploaded wallpaper images (base64, max 8) |
| `display` | JSON | Blur, brightness, overlay opacity, UI opacity |
| `timezone` | string | IANA timezone string (e.g. `Asia/Shanghai`) |
| `pomo-focus` | number | Focus duration (minutes) |
| `pomo-break` | number | Break duration (minutes) |
| `pomo-count` | number | Number of pomodoro sessions |
| `music-mode` | string | Music display mode (`off` / `minimal` / `player` / `immersive`) |
| `music-vol` | number | Music volume (0–1) |
| `music-loop` | string | Loop mode (`sequence` / `single` / `shuffle`) |
| `music-lyrics` | bool | Lyrics display on/off |

## Browser Compatibility

The music player requires the [File System Access API](https://caniuse.com/native-filesystem-api) (`showDirectoryPicker`). Supported in Chrome 86+ and Edge 86+. Firefox is not supported for the music feature; all other features work across modern browsers.
